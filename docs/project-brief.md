# Project Brief — Synthetic Minds Website

## Goal
Build a production-quality website using AI agents (Codex, Claude, GPT) with strong UX, clean code, and secure architecture.

## Setup
- Claude: UX, layout, user flows, design system
- GPT: code generation, debugging, refinement
- Codex: implementation and file structuring

## Stack
- Frontend: Next.js (React) + Framer Motion
- Backend: Supabase or Firebase (managed, secure)
- Auth: Supabase Auth or Clerk

## Rules
- AI generates code, but does NOT make final decisions
- Always review and validate outputs before using
- Do NOT build custom auth from scratch
- Do NOT expose API keys in frontend
- Avoid unverified npm packages

## Security checks
- Input validation on all forms
- Proper authentication flows
- Error handling for edge cases
- Rate limiting where needed

## Expectations
- AI can build a strong MVP quickly
- Final production quality requires human review and iteration

## Workflow
1. Claude designs full UX + structure
2. GPT generates and improves code
3. Codex organizes and implements
4. Human reviews, tests, and secures

## Principle
AI accelerates development, not replaces it.
