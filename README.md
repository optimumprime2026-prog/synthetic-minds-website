# Synthetic Minds Website

Production-focused website project for Synthetic Minds.

## Purpose
Build a production-quality website with strong UX, clean code, and secure architecture using AI-assisted workflows with human review.

## Stack
- Frontend: Next.js (React) + Framer Motion
- Backend: Supabase or Firebase
- Auth: Supabase Auth or Clerk

## Project folders
- `app/` — Next.js app router pages and layouts
- `components/` — reusable UI components
- `lib/` — helpers, configs, integrations
- `public/` — static assets
- `styles/` — global and shared styles
- `docs/` — planning, architecture, briefs
- `notes/` — working notes and review notes
- `prompts/` — prompts for Claude, GPT, Codex
- `agents/` — agent workflow docs and role definitions
- `output/` — generated artifacts and exports
